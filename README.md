# Flogao Photos Backup

Simple .NET Core application to extract and save all photos from Flogao.com.br

Since Flogão is been permanently discontinued on June 29, 2019, I made this tool to extract and backup all photos from any Flogão.

# Usage

You can simple run it using dotnet CLI:
```
dotnet run <FlogaoName>
```

Ex:
```
dotnet run dirrego
```
